LightGBM 문서에 오신 것을 환영합니다!
======================================

**LightGBM** 은 트리 기반 학습 알고리즘을 사용하는 그레이디언트 부스팅 프레임워크입니다. 분산과 효율성을 고려하여 설계되었으며 다음과 같은 장점을 가지고 있습니다:

- 빠른 훈련 속도와 높은 효율성
- 적은 메모리 사용량
- 높은 정확도
- 병렬, 분산 및 GPU 학습 지원
- 대규모 데이터 처리 가능

더 자세한 내용은 `Features <./Features.rst>`__ 을 참고하세요.
